Базові методи показу і приховання

Найпростіші методи. Елемент ховається або з'являється миттєво — або з тривалістю у мілісекундах. Рядок 'slow' = 600мс, 'fast' = 200мс.

show / hide / toggle
$('#box').hide();
$('#box').show();
$('#box').toggle();

// З тривалістю (мс)
$('#box').hide(300);
$('#box').show('slow');
$('#box').toggle(400);

// 02

Fade — зміна прозорості

Елемент плавно з'являється або зникає через зміну opacity. fadeTo() анімує до конкретного значення прозорості.

fadeIn / fadeOut / fadeTo
$('#box').fadeIn(400);
$('#box').fadeOut(400);
$('#box').fadeToggle(300);

// До 50% прозорості
$('#box').fadeTo(400, 0.5);

// Повністю
$('#box').fadeTo(400, 1);
box
// 03

Slide — розкриття як шторка

Елемент плавно розкривається або згортається по висоті. Добре підходить для акордеонів, дропдаунів, списків.

slideDown / slideUp / slideToggle
$('#box').slideDown(300);
$('#box').slideUp(300);
$('#box').slideToggle(400);
Прихований блок Цей елемент з'являється знизу вгору — як шторка.
// 04

.animate() — довільна анімація

Анімує будь-яку числову CSS-властивість. Колір без плагіну не підтримується. Запис += — відносно поточної позиції.

animate()
$('#box').animate({
  opacity: 0.3,
  width:   '120px',
}, 400);

// Відносний зсув
$('#box').animate({
  marginLeft: '+=30px',
}, 300);

// Прогрес-бар
$('#bar').animate({
  width: '100%'
}, 1200);
box
Важливо: .animate() не анімує color і background-color без jQuery UI. Для кольорів краще використовувати CSS transition + toggleClass().
// 05

Черга анімацій і .delay()

jQuery автоматично ставить анімації в чергу — вони виконуються послідовно, одна за одною. .delay() додає паузу між ними.

chaining + delay
$('#box')
  .fadeOut(300)
  .delay(400)
  .fadeIn(300);

// Три анімації поспіль
$('#box')
  .animate({ width: '+=40px' }, 200)
  .animate({ opacity: 0.4 }, 200)
  .animate({
    width: '-=40px',
    opacity: 1
  }, 200);
box

Stagger — поява по черзі

Елементи з'являються не всі одразу, а з невеликою затримкою. Використовуй .each() + setTimeout з множенням на індекс.

staggered fadeIn
$('.item').each(function(index) {
  const $el = $(this);
  $el.hide();

  setTimeout(function() {
    $el.fadeIn(300);
  }, index * 100);
  // ↑ кожен наступний
  //   затримується на 100мс
});

Callback — після завершення

Функція яка виконується після завершення анімації. Критично важлива для видалення елементів — спочатку зникнути, потім видалити з DOM.

fadeOut → remove
// Видалити після зникнення
$('.item').fadeOut(300, function() {
  $(this).remove();
});

// Показати після slideDown
$('#block').slideDown(400, function() {
  $(this).find('span')
    .text('готово!');
});

.stop() — зупинка анімації

Якщо не зупиняти попередню анімацію, нові накопичуються в чергу. Особливо помітно на hover — після швидкого руху мишею елемент довго "доганяє".

без stop() vs з stop()
// ❌ без stop — накопичується
$('#card').on('mouseenter', () => {
  $(this).animate(
    { marginTop: '-6px' }, 200
  );
});

// ✅ зі stop — все чисто
$('#card').on('mouseenter', () => {
  $(this)
    .stop(true, true)
    .animate(
      { marginTop: '-6px' }, 200
    );
});
наведи мишею швидко кілька разів
❌ без stop()
hover me
✅ зі stop()
hover me
stop(true, true) — перший аргумент очищає чергу, другий — завершує поточну анімацію на кінцевому значенні.

Типові прийоми

Готові патерни які зустрічаються в будь-якому проєкті.

toast-сповіщення
function showToast(msg) {
  $('#toast')
    .text(msg)
    .fadeIn(200);

  setTimeout(function() {
    $('#toast').fadeOut(400);
  }, 2000);
}
дія виконана ✦
анімація лайку
$('.like-btn').on('click', function() {
  const $heart = $(this).find('.heart');

  $heart
    .animate({ marginTop: '-6px' }, 100)
    .animate({ marginTop:    '0px' }, 100);

  $(this).toggleClass('liked');
  const n = parseInt(
    $(this).find('span').text()
  );
  $(this).find('span')
    .text(
      $(this).hasClass('liked') ? n+1 : n-1
    );
});
підсвічування помилки форми
$('#btn').on('click', function() {
  const val = $('#input').val().trim();

  if (!val) {
    $('#input')
      .addClass('error')
      .delay(800)
      .queue(function(next) {
        $(this).removeClass('error');
        next();
      });
    return;
  }
  // далі логіка...
});
